# 🚀 System Specs & Performance

This project is optimized and tested on the following **mid-range powerhouse** setup:

| Component | Specification |
| :--- | :--- |
| **CPU** | AMD Ryzen (Mobile) - 8 Cores / 16 Threads |
| **GPU** | NVIDIA GeForce RTX 4060 Laptop GPU (8GB VRAM) |
| **RAM** | 24 GB DDR5 (Dual Channel) |

### ⚡ Performance Expectations
- **Gaming:** Targets **Ultra settings @ 1080p** or **High settings @ 1440p** with DLSS 3 Frame Generation enabled.
- **Productivity:** Handles heavy multitasking, 4K video editing, and compilation workloads with ease.
- **Ray Tracing:** Capable of entry-level ray tracing thanks to the Ada Lovelace architecture.

> *Note: The 24GB RAM configuration offers a sweet spot between the standard 16GB (capacity) and 32GB (cost), ensuring a stable 1% low FPS in modern titles.*

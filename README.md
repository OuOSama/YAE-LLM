# Project YAE — LLM Core Architecture

A high-performance LLM core, reverse-engineered from the **Qwen 3** (Alibaba Cloud). Built to study production-grade neural networks, low-level matrix operations, and GPU efficiency.
---

## 🚀 System Specs & Performance

The core architecture is optimized, benchmarked, and compiled on the following **mid-range powerhouse** environment:

| Component | Specification |
| :--- | :--- |
| **CPU** | AMD Ryzen 7 (Mobile) — 8 Cores / 16 Threads |
| **GPU** | NVIDIA® GeForce RTX™ 4060 Laptop GPU (8GB VRAM) |
| **RAM** | 24 GB DDR5 (Dual-Channel) |

### ⚡ Hardware Benchmarks & Capabilities
* **AI & Productivity:** Seamlessly handles heavy multi-token compilation workloads, matrix dimensions, and 4K creative workflows.
* **Gaming:** Targets stable **Ultra @ 1080p** or **High @ 1440p** utilizing DLSS 3 Frame Generation.
* **Ray Tracing:** Fully capable of entry-level ray tracing powered by the Ada Lovelace architecture.

> 💡 **Engineering Note:** The custom **24GB RAM** configuration strikes the perfect asymmetric sweet spot between standard 16GB (capacity bottleneck) and 32GB (cost efficiency), eliminating heavy memory swapping and securing solid 1% low FPS during extreme multi-tasking and local inference compilation.

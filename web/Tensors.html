🧠 Tensors in Deep Learning

โครงสร้างข้อมูลพื้นฐานที่ขับเคลื่อนโมเดลระดับโลก — Qwen, LLaMA, และทุกอย่างที่ซามะกำลังสร้างอยู่ค่ะ

Tensor vs NumPy Array — Tensor ทำงานบน GPU สำหรับการคำนวณแบบขนานได้เร็วทะลุนรก และรองรับระบบ Autograd สำหรับ Backpropagation อัตโนมัติค่ะ

Rank / มิติของ Tensor — คลิกเพื่อดูรายละเอียด

0D

Scalar

เลขโดดๆ ตัวเดียว ไม่มีมิติ

torch.tensor(5)
1D

Vector

เส้นตรงของตัวเลข (1 แถว)

torch.tensor([1,2,3,4])
2D

Matrix

ตารางสองมิติ (แถว × คอลัมน์)

torch.tensor([[1,2,3],[4,5,6]])
3D

3D Tensor

ลูกบาศก์ตัวเลข ตารางซ้อนกัน

torch.randn(1, 2, 4)
nD

High-Dim

มิติสูง เช่น ข้อมูลภาพ / วิดีโอ

(B, C, H, W)

Shape

(2, 3)

ตาราง 2 แถว × 3 คอลัมน์ ใช้กันทั่วไปใน Weight Matrix ของ Linear Layer ค่ะ เช่น layer ที่รับ input 3 features แล้วแปลงเป็น 2 outputs

มิติ (B, T, C) ใน LLM อย่าง Qwen

dim 1
B
Batch Size
dim 2
T
Seq Length
dim 3
C
Hidden Size

B (Batch Size) — ส่งกี่ประโยคเข้าไปประมวลผลพร้อมกัน เช่น B=32 คือส่ง 32 ประโยคพร้อมกันเพื่อความเร็วใน training ค่ะ

T (Sequence Length / Tokens) — ในประโยคนั้นมีกี่ token เช่น T=2048 คือ context window ของโมเดล

C (Hidden Size / Features) — เวกเตอร์ความหมายของแต่ละ token ใน Qwen2.5-7B ค่านี้คือ 3584 ค่ะ

Common Operations ที่ใช้บ่อยสุดๆ

.shape / .size()

เช็กมิติของ Tensor — สำคัญที่สุดตอน debug ใช้ทุก session ค่ะ

.to(torch.bfloat16)

แปลง dtype เพื่อเซฟ GPU VRAM ใช้ใน training pipeline ของ YAE-LLM

.pow(2)

ยกกำลังสองทุกตัวเลข ใช้ใน RMSNorm variance calculation

.mean(-1, keepdim=True)

หาค่าเฉลี่ยมิติสุดท้าย รักษา shape เดิมไว้สำหรับ broadcasting
